# eks-fargate-profile

This module creates following resources.

- `aws_eks_fargate_profile`
- `aws_iam_role` (optional)
- `aws_iam_role_policy` (optional)
- `aws_iam_role_policy_attachment` (optional)
- `aws_iam_instance_profile` (optional)
